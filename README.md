# slot-auth-service

Identity and Role Management service for the `slot-central` microservices platform — owns a Postgres database and handles authentication/authorization for both **players** (card/UID-based EGM sessions) and **staff/employees**.

This repository is part of a re-architecture of the `slot-central-server-express-rmq` Node.js EGM slot-floor backend into Spring Boot microservices.

Scaffolding in progress.
